# GAME ENGINE WRITTEN IN C
Game engine written in C to learn something new and have some fun!

## What it can do

#### Input
Keyboard Input (KeyDown, KeyUp, KeyPress)
#### Maths
-2D Vectors
#### Renderer
-Draw Textures
-Draw Vectors
-Draw Rectangles
#### Resources
-Load Textures

## How to build

*Open terminal in project root*
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./GAME
```
